# Resume Files

Place your resume PDF here.

## Setup Instructions

1. Save your resume as `resume.pdf` in this folder
2. The download button in your portfolio will link to this file
3. Users can download it with filename: `Arnab_Kumar_Roy_Resume.pdf`

## File Format
- Filename: `resume.pdf` (case-sensitive for GitHub Pages)
- Format: PDF
- Recommended size: < 5MB for faster loading
